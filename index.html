OpenLP to ProPresenter5 converter

This is a very simple python script to convert your OpenLP songs database to ProPresenter. I wrote it as I couldn't get PP5 to import our songs from the text files OpenLP exported.

It also allows you to change the number of lines per slide, which actually means it may be quicker / easier if you need to change slide layout (in terms of how many lines per slide) to keep your song database in OpenLP, and export per event with the appropriate number of lines.

It runs fast, and does our 2700 song database (including Chinese, German and Japanese songs) in about 6 seconds on my iMac.

To use, just run the python script. On OSX, Python is built in, which is helpful.

If you use a terminal, or want to edit stuff / check things / be geeky: cd where_the_python_converter_file_is python converter.py

Some options:

If you edit the script, you should be able to edit a bunch of options (number of lines per slide, default text color, default font, etc) at the top. If it were really needed, more options could probably be added, but to be honest, this does most of what we need.
